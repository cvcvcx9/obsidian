## 설정 방법
1. 루트 폴더에 스프링 어플리케이션 실행 전 필요한 컴포즈 파일 생성
2. application.yml에서 도커 컴포즈 설정
```
spring:
	docker:
		compose:
			enabled: true
```
3. 구현 확인
##### 실행 전 모습
![](https://i.imgur.com/GO7qc6p.png)
- 도커 데스크탑에서 아직 실행이 안된 상태에서의 모습이다
##### 실행 후
![](https://i.imgur.com/BTnuUHW.png)
- 스프링 부트 어플리케이션을 실행하면 자동으로 실행되는 모습을 볼 수 있다
